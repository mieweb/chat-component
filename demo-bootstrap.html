Chat Component Demo - Bootstrap Environment

This demo shows the chat component embedded in a Bootstrap page without style conflicts.

Note: The chat component uses Tailwind CSS with the 'tw-' prefix to avoid conflicts with Bootstrap classes.

Component Features
  • State managed with Zustand

  • Bootstrap-compatible
  • Message callbacks

Usage Example

The component can be embedded in any Bootstrap window or modal without causing style conflicts. All Tailwind classes are prefixed with 'tw-' and Tailwind's base styles are disabled. 

<div id="chat-container"></div>
<script>
  // Initialize component
  const root = ReactDOM.createRoot(
    document.getElementById('chat-container')
  );
  root.render(
    React.createElement(ChatComponent, {
      onMessageSent: handleMessageSent,
      height: '500px'
    })
  );
</script>
